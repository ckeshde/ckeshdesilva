Project Overview

Plugpin is an EV charging station finder mobile app designed to help electric vehicle users quickly locate nearby charging stations, check availability, and navigate easily. The project included full brand identity design and complete UI design for the mobile app.

My Role: UI/UX Designer & Brand Designer
Tools & Stack: Figma, Adobe Illustrator

Problem

EV users often face difficulties when trying to find nearby available charging stations, compare charging speeds, and navigate easily to charging points. Many also feel uncertain about the reliability of unfamiliar charging providers. These challenges highlight the need for a simple, trustworthy, and user-friendly solution that makes the charging experience smooth and stress-free.

Goal

  • Create a clean and modern EV brand identity
  • Design a simple and intuitive mobile experience
  • Help users find charging stations quickly
  • Build trust through consistent branding

Key Findings

  • Many apps are cluttered
  • Poor visual hierarchy
  • Inconsistent branding
  • Complicated filtering systems

UI Design

The final UI was designed to solve the key issues found in competitor apps, such as cluttered screens, poor hierarchy, and complex filters. The app features a clean layout with strong visual hierarchy, making important details like availability and charging speed easy to see. Clear call-to-action buttons and simple navigation help users move through the app quickly and confidently. The filtering system was simplified, and consistent branding creates a trustworthy and seamless experience.

Brand Design

Challenges & Solutions

Challenge: Designing a clean interface for map-heavy screens
Solution: Map screens can easily become cluttered with too much information. To solve this, I simplified the UI by reducing visual noise, using clear icons, and prioritising key information like availability.

Challenge:Balancing functionality with simplicity
Solution: Including essential features without overwhelming users was challenging. I addressed this by simplifying the filtering system and using clear visual hierarchy to guide users naturally through the app.

Outcome

The project resulted in a complete brand identity and a fully designed mobile app interface that delivers a clean, consistent, and user-friendly EV charging experience.
