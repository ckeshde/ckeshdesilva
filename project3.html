Cresuma

Clean web app UI for creating professional resumes

View Design ↗

Project Overview

Redesigned an existing resume-building web application to improve usability, visual appeal, and overall user experience. The goal was to transform a visually boring and confusing interface into a clean, modern, and conversion-focused platform.

My Role:UI/UX Designer & Frontend Developer
Tools & Stack: Figma, React JS & MUI ((Material UI))

Before & After

Old Hero Section

The previous design felt cluttered and difficult to navigate. There were too many CTA buttons, which made it confusing for users to understand what action to take first. The headline didn’t clearly explain the main benefit of the platform, so the value wasn’t immediately obvious. The layout also felt packed and not properly aligned, with everything competing for attention. Overall, the design looked flat and outdated, lacking visual depth and engagement.

New Hero Section

The new design feels much clearer and more intentional. The headline now clearly explains what the platform does and why it’s useful. Instead of multiple competing buttons, there is one strong primary CTA that guides users to start building their resume, along with a secondary option to explore other tools. The layout follows a clear flow, making it easy to read and understand. With better spacing, cleaner alignment, and stronger typography, the interface feels more modern, balanced, and professional.

Challenges & Solutions

Challenge:The previous UI felt outdated, visually dull, and lacked engagement. There was no clear visual hierarchy or emotional appeal.
Solution: Improved typography, spacing, layout, and visual depth to create a cleaner, more modern, and engaging experience.

Challenge: Ensuring the Figma design matched the final React implementation.
Solution: Built reusable MUI components and followed a structured design system for consistency.

Outcome

The redesigned web app turns a cluttered layout into a clean, well-structured experience with improved hierarchy and clearer CTAs. It’s expected to boost engagement by 25–40%, built on a scalable frontend using React JS and MUI.
